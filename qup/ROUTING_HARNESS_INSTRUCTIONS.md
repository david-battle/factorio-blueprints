# QUP routing parameterization gate

Import `routing_parameterization_harness.txt` into Factorio 2.1.11.

This is a development test, not the production QUP. Click **Parameterize** and
confirm that the window starts from concrete Crusher, electric-engine-unit,
steel-plate, and low-density-structure icons rather than purple placeholders.
The five splitters are deliberately recipe-agnostic: they filter Normal through
Legendary quality without specifying an item.

The next production-layout step now depends only on recording which requester
fields Factorio can derive from the Crusher recipe while keeping only the
recipe choice visible during placement. Do not spend time configuring the
fields yet: report whether those concrete icons are present and, if convenient,
send a screenshot of the parameterization window.
